# 樊海宁课题组官网初版

这是 `Fan_LAB` 仓库中的公开官网草稿目录，和内部项目索引、代码规范、数据规则分开管理。

## 文件

- `index.html`：单页静态官网
- `styles.css`：页面样式

## 当前内容

- 首页定位
- PI 简介
- 研究方向
- 公开成果亮点
- 人才培养统计与重绘名单
- 当前成员：樊海宁、铁成炜、姜超
- 公开资料来源
- 联系方式占位

## 后续需要补充

- PI 邮箱、办公室/科室联系方式
- 代表论文完整列表
- 科研项目、平台和获奖信息
- 当前成员照片、研究方向和个人简介
- 经授权可公开的课题组照片或医院/实验室场景图

## 公开边界

官网只放公开展示信息。内部项目路径、患者数据、样本映射表、未发表结果、服务器路径、原始数据和未清理代码不应放入该目录。
