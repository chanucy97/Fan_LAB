# Synthetic example data

`synthetic_tbnk_example.csv` is a simulated, non-patient dataset generated by `src/make_example_data.py`.

It is included only to verify that the analysis code runs and produces the expected output tables. It is not part of the study cohort and does not contain real patients, hospital identifiers, sample identifiers, clinical records, or original flow-cytometry exports.

To regenerate it:

```bash
python src/make_example_data.py --output data/example/synthetic_tbnk_example.csv --n 180 --seed 2026
```
