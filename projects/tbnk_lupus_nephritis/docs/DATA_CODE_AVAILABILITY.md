# Manuscript availability statements

## Data Availability

The raw patient-level and clinical data used in this study are not publicly available because they contain sensitive clinical information and are subject to patient privacy protection, ethics restrictions, and collaborating-institution data governance requirements. De-identified data supporting the main findings may be made available from the corresponding authors upon reasonable request, subject to approval by the corresponding authors, ethics or institutional review, and/or completion of an appropriate data-use agreement.

## Code Availability

The analysis code supporting this study is available in the Fan_LAB GitHub repository under `projects/tbnk_lupus_nephritis`: https://github.com/chanucy97/Fan_LAB/tree/main/projects/tbnk_lupus_nephritis. The repository includes reusable analysis scripts, configuration templates, documentation, and synthetic example data for testing the workflow. Raw patient-level and clinical data are not included in the repository.
