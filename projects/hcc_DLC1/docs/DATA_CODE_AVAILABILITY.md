# Data And Code Availability

## Code

The public code skeleton is available under `projects/hcc_DLC1/` in this repository. The main public script routes private paths through `HCC_DLC1_ROOT`.

## Data

No controlled datasets are included. Public-source data should be downloaded from the relevant portals under their terms of use. Any locally curated clinical tables, single-cell objects, spatial objects, and derived results remain outside this repository.

## Reproduction Boundary

The repository supports reuse of the analysis code when approved users provide the required controlled inputs locally. It does not bundle the data needed to reproduce every figure or result.
